# ai-study-assistants
# AI Study Helper ## 📌 Description of the Project


This project is a study helper that uses AI and was made in Python.  
An AI agent that can also use outside tools helps people answer study-related questions.

The assistant takes in user input and gives structured answers based on the type of request.

 --- ## 🎯 The project's goal

The main goal is to show how an AI agent can:
- get what the user typed
- choose what to do
- use tools like math or finding information
- give back useful results

 --- ## 🧠 Idea for the System

The system is built around one smart agent that:
- looks at the input
- picks a tool if necessary
- gives a final answer

 --- ## Tools Used (planned)

- Tool for calculating  
- Tool for searching  
- Reader for files (optional)
